# Where can I watch it

## How to run locally

If you wanna run it in local, you need a working API key. Add that to your env variables with the name X_RAPIDAPI_KEY.
Also install the packages from reqirements.txt

## About the webapp

Little webapp to learn how to use third part APIs and Flask server.
Insert a the name of a film in the textbox and choose your current region, in order to see where you can stream it!
Deployed at https://wherecaniwatch.onrender.com

Enjoy!
